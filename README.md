# debet

Небольшой учёт доходов и расходов в разрезе проектов.

## Запуск

```powershell
npm install
npm start
```

Откройте приложение в Expo Go на телефоне (отсканируйте QR-код) или запустите эмулятор:

```powershell
npm run android
```

## Экран

Главный экран содержит пять кнопок:

- Создать проект
- Создать расход
- Проекты
- Отчёты
- Настройки
