# PROJECT.md  
## Мобильное приложение Debet - Учет доходов и расходов в разрезе проекта.
---

## 1. Общее описание проекта

«Вуиуе» — это мобильное приложение, предназначенное для Учета доходов и расходов в разрезе проекта.

Приложение:
- работает **полностью офлайн**
- запускается на **мобильном телефоне**
- хранит **все данные локально на устройстве пользователя**
- не требует регистрации и серверной части (MVP)

Основная идея:  
> от ввода первичных документов > к конкретным выводам > к очету и анализу проектов.

---

## 2. Цели приложения

- Хранить список расходов в разрезе проектов
- Сигнализировать о превышении бюджета проекта
- Фиксировать пункты бюджета проекта в реальный анализ по мере выполнения проекта
- Быстро показывать пользователю, **какой следующий пункт бюджета**
- Добавить возможность делать привязку: проект - исполнители (с контактами людей)
---

## 3. Платформа и общие ограничения

- Тип приложения: **мобильное**
- Фреймворк: React Native
- сервис отладки: Expo
- маршруты: Expo Route
- библиотека шаблонов: React Native Paper
- Платформы: Android и iOS
- Хранение данных: SQLite
- Должно работать без интернета
- Геолокация: используется для навигации и карт

---

## 4. Основные режимы работы приложения

Приложение имеет **три ключевых режима**.

---

## 4.1 Режим «Проекты»

### Назначение
Хранилище проектов, которые пользователь хочет выполнить или уже выполнил.


### Сущность: Project (Проект)

Поля:
- `id`
- `name` — название проекта
- `description` — описание
- `visitlater` - boolean, не закончен
- `liked` -  boolean, понравилось
- `dd` - Decimal Degrees, GPS-координаты в десятичном формате
- `photos[]` — список фотографий
- `createdAt` - дата и время создания

### Возможности:
- Добавить новый проект
- Просмотреть список проектов
- Открыть проект на карте
- Прикрепить фотографии
- Использовать проект позже при создании отчета

---


## 5. Навигация внутри приложения

Все компоненты берутся из библиотеки React Native Paper.

Главный экран включает:
- Название программы: Debet
- Вертикальный ряд кнопок:
  - Создать проект
  - Создать расход
  - Проекты
  - Отчеты
  - Настройки

Основные экраны:
- Создать проект
- Детальная карточка проекта
- Создать расход
- Детальная карточка расход
- Проекты
- Отчеты
- Форма отчета
- Настройки

Навигация — простая, мобильная, без вложенных сложных меню.


---

## 6. Хранение данных

- Все данные хранятся **локально**
- Используется одна локальная база данных
- Связи:
  - Projects > Budget > Expenses
- Фотографии хранятся локально, в файловой системе приложения
